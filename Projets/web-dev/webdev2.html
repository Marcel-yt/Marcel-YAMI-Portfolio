Gestion des Tâches en Équipe

Task Manager est une application web collaborative conçue pour optimiser la gestion des tâches au sein des équipes d'entreprise. Cette plateforme permet aux équipes de créer, assigner et suivre les tâches en temps réel, facilitant ainsi la collaboration et la productivité.

Le projet intègre des fonctionnalités essentielles telles que la création de projets, l'attribution de tâches, le suivi des délais, les notifications en temps réel, et un tableau de bord intuitif. L'interface utilisateur moderne et responsive assure une expérience fluide sur tous les appareils.

Technologies utilisées

React.js Laravel 12 MySQL PHP 8 Tailwind CSS Laravel Sanctum REST API Axios
