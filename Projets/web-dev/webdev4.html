Location de maison

Site vitrine fictif spécialisé dans la location de maisons de luxe. Cette plateforme présente un catalogue de propriétés exclusives disponibles à la location pour des séjours de courte ou longue durée.

Le site met en valeur les différentes propriétés avec des galeries photos, des descriptions détaillées, et les caractéristiques de chaque bien. L'interface responsive permet une navigation fluide et intuitive sur tous les appareils.

Technologies utilisées

Figma HTML5 CSS3
